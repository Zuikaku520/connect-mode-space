# Project Overview

## Core Question
Can connection mode be treated as an independent dynamical variable in neural-style computation?

## Central Suspicion
Current models may over-reduce node-to-node connections into a common line plus weight adjustment.

## Project Strategy
This repository does not begin with biological proof.
Instead, it follows a staged computational approach:

1. Minimal mode verification
2. Weight-only replacement testing
3. Combination mode testing
4. Robustness under repeated noisy runs
5. Ablation analysis
6. Synergy map construction
7. Local hotspot scanning

## Current Direction
The main current focus is not consciousness itself, but the possibility that connection mode space may be a missing layer in current cognitive architecture research.
