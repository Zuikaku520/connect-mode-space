# Idea Origin

The original intuition came from optical fiber mode theory.

In optical systems, propagation cannot be faithfully reduced to a simple line or tube.
Different modes, couplings, and propagation structures can produce drastically different outputs.

This raised a question:
if optical propagation cannot be collapsed into a simple line,
why should node-to-node connections in neural-style computation be assumed to be reducible to a common line?

From this analogy emerged the suspicion that current neural modeling may under-model the connection itself.

The project began as a cautious analogy, and only became serious after repeated simulation outputs showed that changing connection mode can induce systematic dynamical differences not replaceable by weight-only adjustment.
