# Current Findings

## 1. Connection mode cannot be ignored
Different connection implementations systematically alter temporal dynamics.

## 2. Connection mode is not reducible to weight-only tuning
Across multiple experiments, mode-induced behaviors remain far from the WeightOnly baseline.

## 3. Higher-order combinations open new dynamical regions
Mode combinations often move the system farther from baseline than their parent modes.

## 4. Synergy is localized, not uniform
Not all parameter regions produce strong synergy.
Synergy emerges only in specific local windows.

## 5. Delayed + Recurrent currently shows the clearest hotspot structure
Two hotspot regions and one control region were locally scanned.
Hotspot A and Hotspot B showed stable positive synergy, while Control C did not. :contentReference[oaicite:2]{index=2}

## 6. The project now supports a "connect mode space" interpretation
Current results suggest that connection modes and their combinations form a structured dynamical space rather than a small set of isolated tricks.

## 7. Hotspots now show measurable geometry
The strongest hotspot regions are not only high-synergy points.
They also show measurable area, connected components, and structured contrast against control regions.
