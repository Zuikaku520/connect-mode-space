# Current Findings

## 1. Connection mode cannot be ignored
Different connection implementations produce systematic differences in output dynamics.

## 2. Connection mode is not reducible to weight-only tuning
Multiple modes remain far from the WeightOnly baseline in standardized feature space.

## 3. Higher-order combinations can open new dynamical regions
Some combined modes move the system farther from baseline than either parent mode alone.

## 4. Synergy is not uniform
Not all combinations produce strong synergy.
Synergy is localized in parameter space.

## 5. Stable synergy windows exist
In current experiments, Delayed + Recurrent exhibits the clearest hotspot windows.

## 6. Control regions matter
Control scans do not show the same level of synergy intensity, suggesting the hotspot effects are structured rather than ubiquitous.
