# Terminology

## Common Line
A simplified edge assumption where connections are treated as generic weighted transmission lines.

## Connection Mode
A structured connection implementation with its own dynamical effects.

## Baseline Region
A parameter region close to the WeightOnly/common-line regime.

## Mid-Shift Region
A region that departs from baseline but does not yet exhibit strong synergy.

## Far-Shift Region
A region that is dynamically far from WeightOnly behavior.

## Synergy Index
A measure of whether a combined mode pushes behavior farther from baseline than its parent modes do individually.

## Synergy Window
A stable local parameter region where synergy remains consistently elevated.

## Parent-Dominated Region
A region where a combination still behaves mostly like one parent mode.
